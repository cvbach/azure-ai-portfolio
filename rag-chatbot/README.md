# Azure AI RAG System (No-Code / Low-Code)

## Overview
This project demonstrates an end-to-end AI system built using Azure services via Azure Portal.

## Architecture
User → Azure AI Search → Azure OpenAI → Response

## Services Used
- Azure AI Search
- Azure OpenAI
- Azure AI Document Intelligence
- Azure AI Content Safety
- Azure AI Foundry

## Key Features
- RAG-based chatbot
- Document ingestion pipeline
- Content moderation

## Implementation
- Built using Azure Portal (no-code approach)
- Data indexed via Azure AI Search
- Integrated using Azure AI Foundry
